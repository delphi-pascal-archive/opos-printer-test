ПРОГРАММА ДЛЯ РАБОТЫ С ПРИНТЕРОМ STARMICRONICS
Проверялась только с TSP100

Написана на Delpxi XE7
код портирован с VISUAL BASIC (С оригинальных исходников идущих вкомплете с принтером)

Для корректной работы обязательно должны стоять оригинальные драйвера от СТАР,
а так же добавлено устройство для режима OPOS.

В папке TPOSComp лежит компонента для работы, если ей что нить понадобиться разберетесь сами
если не разберетесь, ищите другой путь.
